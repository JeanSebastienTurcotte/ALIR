Dans la plupart des cas, on donne une base de l’espace propre associé à chaque valeur propre.
  1. Tous les vecteurs de \(\mathbb{R}^2\) sont des vecteurs propres pour la matrice identité.
  2. Le vecteur \((1,0)\) est un vecteur propre pour la valeur propre \(\lambda_1=1\) et le vecteur \((-1,0)\) est un vecteur propre associé à la valeur propre \(\lambda_2=-1\text{.}\)
  3. Aucun vecteur propre réel.
  4. Pour l’étirement horizontal, le vecteur \((1,0)\) est un vecteur propre associé à \(\lambda_1=r\) et le vecteur \((0,1)\) est un vecteur propre associé à \(\lambda_2=1\text{.}\)
    Pour un étirement vertical, les rôles sont renversés. Le vecteur \((1,0)\) est un vecteur propre associé à \(\lambda_1=1\) et le vecteur \((0,1)\) un vecteur propre pour \(\lambda_2=r\text{.}\)
  5. Tous les vecteurs de \(\mathbb{R}^2\) sont des vecteurs propres.
  6. Le vecteur \((-1,1)\) est un vecteur propre associé à \(\lambda_1=-1\) et le vecteur \((1,1)\) est un vecteur propre associé à \(\lambda_2=1\text{.}\)
  7. Le vecteur \((-w_2,w_1)\) est un vecteur propre associé à \(\lambda_1=0\) et le vecteur \((w_1,w_2)\) est un vecteur propre associé à \(\lambda_2=1\text{.}\)
