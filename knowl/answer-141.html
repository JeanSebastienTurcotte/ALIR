Les quatre types sont:
\begin{equation*} \begin{pmatrix} 1& 0\\ 0& 1 \end{pmatrix}\text{,} \end{equation*}
\begin{equation*} \begin{pmatrix} 0& 1\\ 0& 0 \end{pmatrix}\text{,} \end{equation*}
\begin{equation*} \begin{pmatrix} 1& c\\ 0& 0 \end{pmatrix}\text{ où } c\in \R \end{equation*}
et
\begin{equation*} \begin{pmatrix} 0& 0\\ 0& 0 \end{pmatrix} \end{equation*}
