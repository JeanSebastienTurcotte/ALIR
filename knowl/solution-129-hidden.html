On note \(\text{proj}_{\vec{w}}(\vec{u})\text{,}\) la projection orthogonale du vecteur \(\vec{u}\) sur le vecteur \(\vec{w}\text{.}\) Algébriquement, on a
\begin{align*} \text{proj}_{\vec{w}}(\vec{u}+\vec{v})&=\frac{(\vec{u}+\vec{v})\cdot \vec{w}}{\vec{w}\cdot \vec{w}}\vec{w} && \text{ selon l'équation } \knowl{./knowl/eq-projortho.html}{\text{(1.2.5)}}\\ &=\frac{\vec{u}\cdot \vec{w}+\vec{v}\cdot \vec{w}}{\vec{w}\cdot \vec{w}}\vec{w} && \text{selon la } \knowl{./knowl/prop-propprodscal.html}{\text{propriété}}\text{ de distributivité du produit scalaire}\\ &=\frac{\vec{u}\cdot \vec{w}}{\vec{w}\cdot \vec{w}}\vec{w}+\frac{\vec{v}\cdot \vec{w}}{\vec{w}\cdot \vec{w}}\vec{w}\\ &=\text{proj}_{\vec{w}}(\vec{u})+\text{proj}_{\vec{w}}(\vec{v}) \end{align*}
et
\begin{align*} \text{proj}_{\vec{w}}(k\vec{u})&=\frac{k\vec{u}\cdot \vec{w}}{\vec{w}\cdot \vec{w}}\vec{w} && \text{ selon l'équation } \knowl{./knowl/eq-projortho.html}{\text{(1.2.5)}}\\ &=k\frac{\vec{u}\cdot \vec{w}}{\vec{w}\cdot \vec{w}}\vec{w} && \text{ selon la } \knowl{./knowl/prop-propprodscal.html}{\text{propriété}} \text{ du produit scalaire}\\ &=k\text{proj}_{\vec{w}}(\vec{u})\text{.} \end{align*}
