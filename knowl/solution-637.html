Solution 4.3.4.4.c.1.
En gardant en tête l’exercice précédent, on développe le déterminant voulu selon la première ligne de la matrice \(A_n\text{.}\)
\begin{align*} \det(A_n)&=a_{1,1}C_{1,1}+a_{1,2}C_{1,2}+\cdots +a_{1,n}C_{1,n} & &\text{par } \knowl{./knowl/eq-detcofact.html}{\text{(4.3.5)}}\\ &=2*\det(A_{n-1})-1*(\det(A_{n-1})\text{ avec } a_{1,1}=1)+0-0+ \cdots 0\\ &=\det(A_{n-1})+\det(A_{n-1})-(\det(A_{n-1})\text{ avec } a_{1,1}=1)\\ &=\det(A_{n-1})+(\det(A_{n-1})\text{ avec } a_{1,1}=1 \text{ et } a_{1,2}=0)& &\text{ ar } \knowl{./knowl/li-detsomme.html}{\text{4.2.17:5}}\\ &=\det(A_{n-1})+1*\det(A_{n-2})-0+0+\cdots 0\\ &=\det(A_{n-1})+\det(A_{n-2}) \end{align*}
