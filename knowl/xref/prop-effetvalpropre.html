Proposition 6.1.18. L’effet de certaines opérations sur les valeurs propres.

Soit \(A\text{,}\) une matrice carrée et \(\lambda\text{,}\) une valeur propre. Alors
  1. le scalaire \(r\lambda\) est une valeur propre de la matrice \(rA\text{;}\)
  2. le scalaire \(\lambda^k\) est une valeur propre de la matrice \(A^k\text{;}\)
  3. lorsque \(A\) est inversible, \(\frac{1}{\lambda}\) est une valeur propre de la matrice \(A^{-1}\text{;}\)
  4. la valeur propre \(\lambda\) est aussi une valeur propre de transposée \(A^T\text{.}\)

Démonstration.

Soit \(\vec{v}\text{,}\) un vecteur propre associé à \(\lambda\) pour la matrice \(A\text{.}\) On a
\begin{align*} (rA)\vec{v}&=r(A\vec{v}) && \text{associativité du produit scalaire matrice }\\ &=r\lambda \vec{v}\text{.} \end{align*}
Ainsi, puisque \((rA)\vec{v}=(r\lambda)\vec{v}\text{,}\) une valeur propre de \(rA\) est \(r\lambda\text{.}\) De plus, le vecteur \(\vec{v}\) demeure un vecteur propre.

Démonstration.

Si \(A\) est une matrice inversible, alors selon le Théorème de la matrice inverse, sixième version, la valeur propre n’est pas nulle. Soit \(\vec{v}\text{,}\) un vecteur propre associé à \(\lambda\text{.}\) On peut alors écrire
\begin{align*} A^{-1}\vec{v}&= A^{-1}\frac{\lambda \vec{v}}{\lambda}\\ &=A^{-1}\frac{A\vec{v}}{\lambda} && \text{ car } A\vec{v}=\lambda\vec{v}\\ &=A^{-1}A \frac{1}{\lambda}\vec{v}\\ &=\frac{1}{\lambda}\vec{v}\text{.} \end{align*}
Le scalaire \(\frac{1}{\lambda}\) est donc bel et bien une valeur propre de \(A^{-1}\text{.}\)
